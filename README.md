# ensolvers-todo

## Requirements

- PostgreSQL >= 12.9
- Node.js >= 17.8.0
- NPM >= 8.5.5

## Tech used

- Nest.js == 8.0.0
- Prisma == 8.0.0
- React.js == 17.0.2
- Mui == 5.5.3

## Installation guide

1. Clone this repository:

```
git clone https://github.com/rbestardpino/ensolvers-todo.git
```

2. Go to the project folder and run the start script:

```
bash start.sh
```

3. Use the app by opening http://localhost:3001 in a browser

## Tested in:

- Windows 10
- Ubuntu 20.04
